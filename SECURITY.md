# Security Policy

This repository contains public showcase materials only and should never contain:

- API keys, tokens, passwords, or credentials
- private source code
- proprietary prompts or decision logic
- exchange or brokerage account information
- internal network details
- customer or personal data

If you discover sensitive information in this repository, do not repost it or open a public issue containing the data. Contact the repository owner privately through the GitHub profile associated with this repository.
